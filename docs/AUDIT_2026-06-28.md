### C-01: Нет `recover()` ни в одной горутине (lines 1493, 1521-1523, 1656-1657)

**Файл:** `server/server.go`  
**Описание:** 5 горутин без `defer recover()`: сигнальный хэндлер (1493), `statsLoop` (1521), `expiredPasswordJanitor` (1522), `botLoop` (1523), `proxyToWG`/`proxyToClient` (1656-1657). Любая паника убивает весь сервер.

### C-03: Нет очистки iptables/nft при shutdown — утечка правил

**Файл:** `server/server.go` (lines 1270-1340)  
**Описание:** `setupFullConeNAT()` добавляет правила MASQUERADE и FORWARD, но нет функции `cleanupNAT()`. При завершении сервера правила остаются.

**Фикс:** Добавить `cleanupNAT()` и вызвать в defer:
```go
// Добавить в main():
defer func() {
    cleanupNAT(wgIfaceName)
    wgDev.Close()
    runCmdSilent("ip", "link", "del", wgIfaceName)
}()

func cleanupNAT(wgIface string) {
    switch {
    case commandExists("iptables"):
        ipt := exec.Command("iptables-save")
        grep := exec.Command("grep", "-v", "WDTT_MANAGED")
        rst := exec.Command("iptables-restore")
        grep.Stdin, _ = ipt.StdoutPipe()
        rst.Stdin, _ = grep.StdoutPipe()
        ipt.Start()
        grep.Start()
        rst.Run()
    case commandExists("nft"):
        exec.Command("nft", "delete", "table", "ip", "wdtt").Run()
    }
}
```

func sanitizeDeviceID(id string) string {
    if len(id) > 64 {
        id = id[:64]
    }
    // Только a-z, A-Z, 0-9, -, _
    return strings.Map(func(r rune) rune {
        if (r >= 'a' && r <= 'z') || (r >= 'A' && r <= 'Z') || (r >= '0' && r <= '9') || r == '-' || r == '_' {
